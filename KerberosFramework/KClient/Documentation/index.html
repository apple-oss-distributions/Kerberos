KClient 3.0
What is KClient 3.0?

KClient 3.0 is an improved KClient API for the Macintosh. KClient 3.0 is available as part of the Kerberos framework on Mac OS X.

Where is the API specification?

Please refer to the KClient 3.0 API for a list of the available functions and how to use them.

For assistance porting your existing KClient 1.x applications to 3.0, check out the KClient Transition Guide.

Where do I get KClient 3.0?
KClient ships as part of Kerberos for Macintosh included with Mac OS X. The latest release is included with Mac OS X 10.2 and later.

Please read the KClient Library Bugs List and Version History for information on the latest release of KClient.
