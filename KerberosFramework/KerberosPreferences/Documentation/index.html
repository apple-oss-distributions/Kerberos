PreferencesLib
What is PreferencesLib?

PreferencesLib is a CFM shared library for use with MIT Kerberos for Macintosh. It provides APIs for accessing the Kerberos Preferences file.

Where is the API specification?

You can find the API documentation here.

Where do I get PreferencesLib?

The PreferencesLib SDK is part of MIT Kerberos for Macintosh. You can get the latest release of the MIT Kerberos for Macintosh here.

Please read the PreferencesLib Bugs List and Version History for information on the latest release of PreferencesLib.
